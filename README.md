## Technologies Used

- Amazon Web Services (AWS) - EC2, S3, IAM, VPC, Lambda, CloudFormation, RDS, and more
- Infrastructure as Code (IaC)
- Cloud Security best practices
- Automation and orchestration tools

## Project Highlights

- Designed and deployed a secure, scalable AWS infrastructure
- Automated resource provisioning using CloudFormation templates/scripts
- Implemented IAM roles and policies to enforce least privilege access
- Configured storage solutions with backups and lifecycle policies
- Monitored system health using CloudWatch and alarms
